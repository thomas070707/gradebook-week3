# gradebook

과목별 점수를 입력받아 성적을 관리하는 프로그램입니다.

## 기능
- 과목별 점수 입력
- 평균 점수 계산
- 성적 결과 출력
- 최고점 / 최저점 과목 확인
- 점수 → 학점 변환 (A+~F)
- 점수 파일 저장 / 불러오기
- 점수 통계 (분산, 표준편차)

## 사용법
```
python grades.py
```

## 실행 예시
```
과목별 점수를 입력하세요 (0~100).

국어 점수: 88
영어 점수: 92
수학 점수: 75
과학탐구 점수: 81

==============================
       성적 결과
==============================
  국어           88.0점
  영어           92.0점
  수학           75.0점
  과학탐구        81.0점
------------------------------
  평균           84.0점
==============================

최고점: 영어 (92.0점)
최저점: 수학 (75.0점)
```

## 파일 구성
- `grades.py` — 점수 입력, 평균 계산, 결과 출력
- `converter.py` — 점수 → 학점 변환
- `file_io.py` — 점수 파일 저장 / 불러오기

## 개발 환경
